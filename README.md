## How to install and run locally ?

```
$ git clone https://github.com/fahd23/CricBazzar-ecom-react.git
$ cd CricBazzar-ecom-react
$ npm install
$ npm start
```

---

## Features?
- Home Page
- Product Listing Page
- Filters by Price, Category and Ratings
- Cart Management
- Wishlist Management
